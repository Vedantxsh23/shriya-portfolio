Autonomous Waste Collection Robot with Robotic Arm

  • Designed and developed a mobile robot equipped with a robotic arm to pick and place solid waste into an onboard collection container.
  • Integrated ultrasonic sensors and motor control systems for object detection, movement control, and automated pick-and-place operations.
  • Implemented a rotating brush-based dust cleaning mechanism to collect fine particles while the robot navigates.
  • Tech Stack: Arduino, Embedded C/C++, DC Motors, Servo Motors (MG996R), Ultrasonic Sensor (HC-SR04), L298N Motor Driver.

Natural Language-Driven Agentic Web Information Retrieval System

  • Built an AI agent that autonomously navigates multiple web pages to gather and consolidate information, mimicking how users browse through multiple tabs.
  • Used LLM-based web scraping to extract domain-specific data from websites and synthesize final answers from dispersed sources.
  • Integrated multimodal processing to analyze both text and image content, enabling richer and more context-aware information retrieval.

UniGO — AI-Powered Professor Discovery Platform

  • Built a web app to help students find research professors at IITs/NITs via semantic interest matching. Scraped and structured data of 2600+ professors across 16 IITs and 55+ departments with verified emails.
  • Built a match scoring algorithm with semantic 65% and keyword 35% weighting, and deployed a responsive 4-page UI on GitHub Pages. The platform is still under active development.
  • Tech Stack: HTML5, CSS3, Vanilla JavaScript, Python, pandas, openpyxl, GitHub Pages.
